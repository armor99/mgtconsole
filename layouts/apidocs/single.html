/user/{userID}/setpw

This endpoint lists all users for a particular customer. It will implement paging on the return output.

Path Parameters
The path parameter, userID, is the userID for the user that is setting their password.

Header Parameters
Content-Type application/json

Body Parameters
customer_id Armor Identity assigned customer ID number Required
password User’s password string Required
pw_hash Password hash generated by created a new user or resetting a user’s password string Required

Response Parameters
As described on the XXXX page, responses always contain a 'status' object, and optional contain 'data' and 'paging' objects. The 'data' object for this endpoint contains the follow parameters:
access_token JWT used to access a restricted resource. string
refresh_token JWT used to obtain a new Access Token after current token expires. string
